Prelusion
=========

We will have some discussion in *Maximum Subsequence Sum*. The main question is
to solve: For a given sequence :math:`A_k`, evaluate the function

.. math::

   f(i,j) = \max{\left\{\sum_{k=i}^{j} A_{k}\right\}}.
