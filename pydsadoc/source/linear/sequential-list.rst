.. rubric:: Stack

A stack can be implemented with a sequential list :py:class:`SeqList` and its
methods :py:meth:`SeqList.append` and :py:meth:`SeqList.pop` described above.
Especially, the time complexity of the two methods is :math:`O(1)`.
